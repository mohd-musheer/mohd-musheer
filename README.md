## Hello, I’m Mohd

I’m an AI/ML engineer and BCA student at Takshashila Mahavidyalaya, Amravati (2023–2026), based in Amravati, Maharashtra, India. I build practical machine-learning systems—from NLP workflows and private RAG assistants to LLM tooling and performance-oriented ML infrastructure.

## ◈ What I build

- **Retrieval and knowledge systems** that turn documents, profiles, and technical documentation into useful, grounded assistants.
- **LLM applications and tooling** with APIs, streaming workflows, semantic understanding, and deployment in mind.
- **ML and NLP products** that make ranking, analysis, and automation more transparent and usable.
- **AI infrastructure experiments** focused on efficient inference, modular architecture, and systems-level performance.
